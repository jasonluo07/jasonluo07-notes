「data-\*」屬性用於在 HTML 元素上儲存自定義的額外資訊。這些屬性允許我們在不干擾標準 HTML 結構和行為的情況下，將資料綁定到特定元素上。這些資料可以被 JavaScript 輕鬆讀取，用於實現頁面的特定功能或交互。使用「data-」屬性的主要優點是它保持了 HTML 的清晰和語義化，同時為前端代碼提供了更大的靈活性和可維護性。
